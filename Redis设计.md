1. Simply Dynamic String 简单动态字符串

   - 除了保存数据库中的字符串值，还作为缓冲区，比如AOF，输入缓存区

   - ```c++
     struct sdshdr
     {
     	int len; //长度
     	int free; //剩余长度
         char buf[] //char数组
     }
     ```

     与C++ 字符串区别

     可以直接获取字符串长度

     杜绝缓冲区溢出

     减少修改字符串带来的内存分配

   - 空间预分配

     - 如果SDS长度小于1M，则长度翻倍
     - 如果SDS长度大于1M，多分配1M

   - 惰性空间释放

     - 如果长度变短，不先释放空间
