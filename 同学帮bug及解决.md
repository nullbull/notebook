3. `No message found under code 'productname.required' for locale 'null'.`说明国际化配置文件出现了问题，没有配置好，访问不到该配置文件的code，需要在yml添加

   ```
   spring:
     messages:
         #国际化资源文件路径
       basename: i18n/messages
   ```

一定要控制好yml文件的格式，不然会无效，验证方法是ctrl+B能够访问到
